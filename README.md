# pwet-image-loader

A [Pwet](https://github.com/Gastonyte/pwet) component to preload a set of images
 
### Show builded example

Just open :

```example/build/index.html```

### Build example

```
yarn && yarn demo
```
or
```
npm i && npm run demo
```
Now the demo is visible at [http://localhost:3000](http://localhost:3000)
